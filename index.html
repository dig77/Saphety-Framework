Good design systems can help you create digital products with efficiency and consistency. But great design systems will support and strengthen your team’s creativity at the same time.

A design system is a tool. We can think of it as a collection of reusable components that are guided by a set of standards. You can then use this collection to build any number of applications.

A design system is a collection of assets and a set of standards and documentation that accompanies that collection. A pattern library is a collection of components and guidelines for using them. A style guide is a set of guidelines for the graphic styles, typefaces, colours, images and so on. In a design system, you'll find examples of how to use each component or graphic style along with accessibility information, best practices and guidelines.

Today, the talent, techniques, thinking and tools of design, like technology, have become more and more abundant and accessible to all alike. Paradoxically, decisions are more distributed, and experiences can become less and less distinctive. The rapid, relentless pace to produce can create an uneven emphasis on immediacy and incrementalism over individuality and innovation.

It is more important than ever that we own our own ethos, make palpable our brand values, and incorporate an instantly identifiable SAPHETYness in everything we do.

Given our scale and scope, ‘good design’ is not only a requirement, it’s a deeper responsibility to relationships we seek to serve.
